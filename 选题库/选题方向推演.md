# 从底层原理长出选题方向

---

## 为什么"从底层往上长"是有效的

80+ 个成功小程序，品类不同（接龙、签到、班级、工时），但底层原理完全一致：

```
原理 ①  找到一个"组织者角色"——这个人有反复出现的管理负担
原理 ②  工具必须在群里使用——使用即传播，参与者被迫激活
原理 ③  组织者为"省时间+可追溯+显得专业"付费
原理 ④  高频重复（每天/每周）——形成依赖后切换成本高
```

接龙符合这 4 条，所以成了。签到符合这 4 条，所以也成了。班小二符合这 4 条，所以做到 9 个子产品。小时工记工时稍微变了一下——组织者变成了"自己管理自己"，但自我保护的驱动力同样强。

**原理一致 = 成功的底层条件一致。** 所以方法不是"想一个新功能"，而是：

> 找到还没被工具化的"组织者 × 高频协作"场景，然后问：AI 在这里能做到什么以前做不到的事？

---

## 推演方法

```
第一步：列出中国社会中所有的"组织者角色"
第二步：筛选——哪些组织者有高频、重复的群协作需求
第三步：检查——这些需求现在是靠聊天/纸笔/Excel 凑合的吗？
第四步：叠加——AI 在这里能创造什么"以前做不到"的价值？
第五步：验证——符合 8 项选题标准吗？
```

## 第一步：中国社会中的"组织者角色"全景

| 组织者 | 管理的群体 | 典型群规模 | 已有工具化程度 |
|--------|-----------|----------|-------------|
| 班主任/老师 | 家长+学生 | 30-50人 | **高**（班小二、云班级、接龙） |
| 班委/家委 | 本班家长 | 30-50人 | **高** |
| 社区团长 | 邻居/顾客 | 50-500人 | **中**（接龙+收款工具拼凑） |
| 公司部门主管 | 下属 | 5-30人 | **中**（企业微信/钉钉覆盖） |
| 业主委员会 | 小区业主 | 100-500人 | **低** |
| 兴趣社群群主 | 爱好者 | 50-500人 | **低** |
| 健身/减脂群领队 | 学员 | 20-100人 | **低** |
| 读书会/学习群组织者 | 成员 | 10-50人 | **低** |
| 家族群管理者 | 家人 | 10-50人 | **低** |
| 宝妈群活跃家长 | 同龄孩子家长 | 20-200人 | **低** |
| 培训班老师/教练 | 学员 | 10-50人 | **低** |
| 自由职业者/兼职群管理 | 同行 | 50-500人 | **低** |
| 慢病管理者（家属） | 患者+家人 | 3-10人 | **极低** |
| 老人家庭照护协调者 | 兄弟姐妹 | 3-8人 | **极低** |

**观察**：工具化程度高的场景（班级、接龙）已经是红海。**工具化程度低的场景才是机会。**

## 第二步+第三步：筛选出"高频 + 还在靠聊天凑合"的场景

筛掉已有成熟工具的、筛掉低频的、筛掉群传播弱的，剩下这些：

### 场景 A：家长辅导作业（每天）

```
现状：
  老师在群里布置作业 → 家长辅导孩子做 → 拍照发群/交小程序
  家长的真正痛点不是"交作业"，而是"辅导不了"

组织者：家长自己（自我管理型）
频率：每天
群传播：家长群互相推荐
现有解决方案：要么花钱报班，要么自己硬撑，要么放弃
```

### 场景 B：健身/减脂/习惯养成群的领队（每天）

```
现状：
  群主/教练每天要：收打卡 → 逐个点评 → 鼓励坚持 → 踢不打卡的
  打卡工具已有（群打卡），但"点评+鼓励+个性化建议"全靠人工

组织者：群主/教练/领队
频率：每天
群传播：天然——打卡本身就在群里
现有解决方案：人工逐个回复，或者不回复（导致群死掉）
```

### 场景 C：培训班/私教/小班课老师的学员管理（每天-每周）

```
现状：
  课后要：发练习 → 收练习 → 给反馈 → 通知下节课 → 收费
  现有工具只解决了"发+收"，"反馈"还是手动的

组织者：老师/教练
频率：每次课（每天-每周）
群传播：学员群/家长群
现有解决方案：微信消息+班小二/接龙拼凑
```

### 场景 D：慢病自我管理 + 家属协作（每天）

```
现状：
  糖尿病/高血压/痛风患者每天要：记饮食 → 记数据 → 调方案
  家属想了解但患者嫌烦
  去医院复查时医生要看记录，但记录散落在各处

组织者：患者自己 或 家属（替父母管理）
频率：每天（记录）+ 每周/月（复查）
群传播：病友群、家庭群
现有解决方案：纸笔记录、Excel、零散的健康App
```

### 场景 E：家族群管理者——家庭记忆/大事协调（每周）

```
现状：
  谁过生日、谁的孩子考试、年夜饭怎么安排、老家房子怎么处理
  全靠群聊刷屏，重要信息淹没在表情包里

组织者：家族里"操心的那个人"（通常是长女/长嫂）
频率：每周有小事，每月有大事
群传播：家族群
现有解决方案：完全没有——所有信息都在聊天记录里
```

### 场景 F：社群团长/私域运营者的内容生产（每天）

```
现状：
  每天要：选品 → 写推荐文案 → 发群 → 处理咨询 → 统计订单
  选品和文案最耗时间，但每天都要做

组织者：团长/群主
频率：每天
群传播：商品分享本身就在群里
现有解决方案：手动写文案、复制粘贴
```

## 第四步：叠加 AI——哪里能创造"以前做不到"的价值？

不是"给工具加个 AI 按钮"，而是问：**有了 AI，这个场景的核心体验能发生质变吗？**

| 场景 | AI 创造的质变 | 质变程度 | 以前为什么做不到 |
|------|-------------|---------|---------------|
| A. 辅导作业 | 拍照 → AI 逐题批改 + 讲解错因 + 生成同类练习 | **极高** | 需要老师水平的人逐题看 |
| B. 打卡点评 | 用户交打卡内容 → AI 自动生成个性化点评+建议 | **高** | 领队不可能每天给 50 人写点评 |
| C. 学员反馈 | 学员交练习 → AI 批改 + 生成进步报告 | **高** | 老师时间有限，反馈经常缺失 |
| D. 慢病管理 | 记录饮食/数据 → AI 分析趋势 + 给出调整建议 | **高** | 需要营养师/医生的专业判断 |
| E. 家族协调 | 群聊中 → AI 自动提取待办/纪念日/决议 | **中** | 信息提取需要理解上下文 |
| F. 社群文案 | 商品图片 → AI 自动生成推荐文案 | **中高** | 写好文案需要营销能力 |

## 第五步：6 个具体选题方向

### 方向 1：拍作业 — AI 家庭教师

**一句话**：家长拍孩子作业，AI 秒批改、讲错因、出同类题巩固。

```
底层原理对照：
  ✅ 组织者 = 家长（管理孩子学习的人）
  ✅ 高频 = 每天
  ✅ 群传播 = 家长群互相推荐"这个工具好用"
  ✅ AI 质变 = 以前要花钱请家教/报班才能做到的事，现在拍一下就行

核心动作：拍照 → 看批改结果 → 让孩子改正 → 做巩固题
付费点：免费 3 次/天，VIP 无限次 + 错题本 + 周报
用户画像：25-40岁家长，孩子小学1-6年级，每月 19.9 元
人群规模：中国小学生约 1.07 亿，对应 1 亿+ 家长
```

**为什么以前不存在**：OCR + 大模型结合，才能做到"看懂手写作业 + 理解解题过程 + 判断对错 + 解释原因"。两年前的技术做不到这件事。

---

### 方向 2：AI 打卡教练 — 群打卡的智能点评官

**一句话**：健身/减脂/学习群里，成员打卡后 AI 自动生成个性化点评和建议，领队省 90% 的时间。

```
底层原理对照：
  ✅ 组织者 = 群主/教练/领队
  ✅ 高频 = 每天
  ✅ 群传播 = 打卡+点评本身就在群里发生
  ✅ AI 质变 = 以前领队手动回复 50 个人要 1 小时，现在 AI 秒回

核心动作：成员打卡（文字/图片）→ AI 即时生成点评 → 群内可见
付费点：领队付费（免费管 20 人，VIP 管 200 人 + 周报 + 数据分析）
用户画像：健身教练/减脂营领队/读书会群主，每月 29.9 元
人群规模：中国健身人口 3 亿+，私域社群运营者千万级
```

**为什么以前不存在**：给 50 个人写 50 条不同的、有针对性的点评——这件事以前只有人能做，而且大部分领队也懒得做。AI 让"每个人都被看见"变得可能。

---

### 方向 3：AI 练习反馈 — 课后教练

**一句话**：培训班/私教学员交练习（拍照/录音/录像），AI 自动批改+生成进步报告，老师只需看报告确认。

```
底层原理对照：
  ✅ 组织者 = 培训老师/教练
  ✅ 高频 = 每次课后（每天-每周）
  ✅ 群传播 = 学员群/家长群
  ✅ AI 质变 = 练琴/练书法/练口语，以前课后就没人指导了

核心场景（先做 1 个垂直切口）：
  - 钢琴/乐器：录音 → AI 分析节奏/音准/表现力
  - 书法/写字：拍照 → AI 分析笔画/结构/进步
  - 英语口语：录音 → AI 评测发音/流利度/建议
  - 体育动作：视频 → AI 分析姿势（俯卧撑/瑜伽等）

付费点：老师付费（管理学员）+ 家长付费（看孩子进步报告）
用户画像：培训机构老师/私教，每月 39.9 元
```

**为什么以前不存在**：多模态大模型才能"看懂"图片/音频/视频中的练习质量。这个能力是 2024-2025 年才成熟的。

---

### 方向 4：AI 饮食管家 — 慢病人群的每日助手

**一句话**：拍一下吃的东西，AI 告诉你能不能吃、吃多少合适、今天总摄入怎么样，替代营养师的日常指导。

```
底层原理对照：
  ✅ 组织者 = 患者自己 或 替父母管理的子女
  ✅ 高频 = 每天 3 餐
  ✅ 群传播 = 病友群分享 + 家庭群（子女推荐给父母）
  ✅ AI 质变 = 以前请营养师 200-500 元/次，现在每餐拍照就行

核心动作：拍餐食 → AI 识别食物 → 结合用户病情给建议 → 记录当日摄入
付费点：免费 1 餐/天，VIP 不限次 + 周报 + 复查报告生成，19.9 元/月
用户画像：
  - 直接用户：40-65岁慢病患者（糖尿病/高血压/高尿酸/肾病）
  - 代管用户：25-40岁子女替父母用
人群规模：中国糖尿病患者 1.4 亿，高血压 2.45 亿
```

**为什么以前不存在**：食物识别 + 营养分析 + 结合个人病情给建议——这三步串起来，以前需要一个营养师面对面做。大模型的多模态识别 + 医学知识 + 个性化推理，第一次让这件事变成拍一张照片就能完成。

---

### 方向 5：AI 团长文案机 — 私域卖货的内容引擎

**一句话**：团长拍商品/贴商品链接，AI 秒生成适合群发的种草文案，每天省 2 小时写文案的时间。

```
底层原理对照：
  ✅ 组织者 = 社区团长/私域群主/微商
  ✅ 高频 = 每天发 3-10 条
  ✅ 群传播 = 文案本身就是发在群里的
  ✅ AI 质变 = 以前靠复制品牌方素材（千篇一律）或自己写（写不好）

核心动作：拍商品/输入商品信息 → 选风格 → AI 生成多版文案 → 一键复制发群
付费点：免费 3 条/天，VIP 无限 + 多风格 + 配图生成，29.9 元/月
用户画像：社区团长/微商/私域运营者
人群规模：中国社区团购从业者数百万，私域运营者千万级
```

**为什么以前不存在**：写出"像真人推荐而不像广告"的文案，以前需要文案能力。大模型可以根据商品特点 + 群的调性 + 之前的爆款风格，生成"接地气"的文案。

---

### 方向 6：AI 错题诊断 — 考后分析师

**一句话**：拍试卷 → AI 识别错题 → 归因到知识点 → 生成个人薄弱点地图 → 推荐针对性练习。

```
底层原理对照：
  ✅ 组织者 = 家长（管理孩子学习）
  ✅ 高频 = 每周（周测/单元测/月考后）
  ✅ 群传播 = 家长群推荐
  ✅ AI 质变 = 以前只有好老师能做"错题归因分析"，大部分家长只看分数

核心动作：拍错题/试卷 → AI 归因 → 看薄弱知识点 → 做针对练习
付费点：免费分析 2 次/月，VIP 无限 + 错题本 + 趋势追踪，29.9 元/月
用户画像：小学3年级-初中3年级学生家长
人群规模：中国中小学生约 1.8 亿
```

**为什么以前不存在**：判断"这道题错了是因为哪个知识点没掌握"——这需要学科理解能力。以前只有经验丰富的老师能做，而且不可能给每个学生做。

## 6 个方向的优先级排序

| 维度 | ①拍作业 | ②打卡教练 | ③练习反馈 | ④饮食管家 | ⑤团长文案 | ⑥错题诊断 |
|------|--------|---------|---------|---------|---------|---------|
| AI 质变程度 | 极高 | 高 | 高 | 高 | 中高 | 极高 |
| 用户规模 | 1亿+ | 千万 | 百万 | 2亿+ | 千万 | 1亿+ |
| 付费意愿确定性 | 极高 | 中 | 高 | 高 | 高 | 极高 |
| 一个人能做 MVP | 是 | 是 | 需选垂直 | 是 | 是 | 是 |
| 群传播强度 | 强 | 极强 | 中 | 中 | 强 | 强 |
| 竞争态势 | 有先行者 | 空白 | 空白 | 有但体验差 | 有但不好用 | 有先行者 |
| **综合优先级** | **A** | **A** | **B** | **A** | **B** | **A** |

### A 级（建议优先评估）

- **①拍作业**和**⑥错题诊断**——可合并为一个产品的两个功能，面向同一用户（家长），同一痛点（辅导不了+不知道孩子薄弱在哪）
- **②打卡教练**——最符合"群管理者工具"模型，群传播最强，且几乎没有竞品
- **④饮食管家**——人群最大（慢病人群 3 亿+），付费动机最硬（健康焦虑），且可以通过"子女帮父母用"实现跨代传播

### B 级（值得探索）

- **③练习反馈**——需要选一个垂直切口做深，比如"练字"或"练琴"
- **⑤团长文案**——需求真实，但文案生成的壁垒不高，容易被模仿

## 底层原理一致性校验

最后回到开头的问题：**为什么从底层往上长是有效的？**

```
接龙 的成功公式：
  群管理者 × 每天收集信息 × 发到群里就是传播 × 省时间值得付费

拍作业 的成功公式：
  家长 × 每天辅导作业 × 家长群推荐 × 替代家教值得付费

打卡教练 的成功公式：
  群领队 × 每天点评打卡 × 打卡本身在群里 × 省时间值得付费

饮食管家 的成功公式：
  慢病患者/子女 × 每天 3 餐 × 病友群/家庭群推荐 × 替代营养师值得付费
```

**公式结构完全一致。** 变的只是"谁""管什么""群在哪"。

而我们叠加的 AI 变量，让每个公式多了一项：

> **× 以前这件事需要专家才能做，现在 AI 做到了**

这就是选题标准第 2 条（依赖大模型实现本质突破）与已验证模型的结合点。
