# AI智能人群食谱小程序 - 原型设计稿

## 页面结构

```
小程序主要页面：
├── 1. 欢迎页
├── 2. 用户画像采集页
├── 3. 首页（今日推荐）
├── 4. 食谱详情页
└── 5. 我的页面
```

---

## 页面 1: 欢迎页

### 布局

```
+-------------------------------------------+
|                                           |
|         [LOGO: 智能食谱]                   |
|                                           |
|      根据你的健康需求                     |
|      AI生成专属食谱                       |
|                                           |
|                                           |
|     [ 开始使用 ] (大按钮，主色)            |
|                                           |
|                                           |
+-------------------------------------------+
```

### 设计要点
- 简洁干净，突出核心价值
- "开始使用"按钮醒目，引导用户进入采集流程
- 底部可加一行小字："科学食谱 · 营养师审核"

## 页面 2: 用户画像采集

### 分步流程

#### Step 1: 基础信息

```
+-------------------------------------------+
|  < 返回                    步骤 1/3        |
|                                           |
|   基础信息                                |
|                                           |
|   性别    [ ○ 男  ○ 女 ]                  |
|   年龄    [ 28 ]                          |
|   身高    [ 165 ] cm                      |
|   体重    [ 55 ]  kg                      |
|                                           |
|              [ 下一步 ]                  |
+-------------------------------------------+
```

#### Step 2: 健康标签（核心）

```
+-------------------------------------------+
|  < 返回                    步骤 2/3        |
|                                           |
|   选择你的健康需求                        |
|                                           |
|   [ ○ ] 孕期女性                          |
|         若选择，展开：孕周 [下拉选择]     |
|                                           |
|   [ ○ ] 糖尿病                            |
|   [ ○ ] 高血压                            |
|   [ ○ ] 健身                              |
|         若选择，展开：[ ○ ] 减脂  [ ○ ] 增肌 |
|   [ ○ ] 失眠                              |
|                                           |
|   （单选）                                 |
|                                           |
|              [ 下一步 ]                  |
+-------------------------------------------+
```

**设计说明**：
- 健康标签单选，便于AI生成精准食谱
- 选中某个标签后，展开二级选项（如孕周、减脂/增肌）
- 慢性病标签选择后，右下角弹出小气泡："本工具仅供参考，不能替代医生建议"

#### Step 3: 饮食禁忌

```
+-------------------------------------------+
|  < 返回                    步骤 3/3        |
|                                           |
|   饮食禁忌（可选）                        |
|                                           |
|   选择你不吃的食材：                      |
|   [ ] 海鲜                                |
|   [ ] 牛肉                                |
|   [ ] 辛辣食物                            |
|   [ ] 乳制品                              |
|   [ ] 蛋类                                |
|   [ ] 羊肉                                |
|   [ ] 自定义 [ + ]                        |
|                                           |
|   已选：海鲜、牛肉                        |
|                                           |
|     [ 生成我的食谱 ] (大按钮)             |
+-------------------------------------------+
```

**交互说明**：
- 点击标签选中，再次点击取消
- 右上角实时显示已选标签
- "自定义"按钮支持用户输入其他禁忌食材

## 页面 3: 首页（今日推荐）

### 布局

```
+-------------------------------------------+
|   我的        今日推荐      食谱库         |
+-------------------------------------------+
|                                           |
|  今日 · 2026-03-05  星期四                |
|                                           |
|  ┌─────────────────────────────────────┐|
|  │ 早餐                                │|
|  │ 燕麦牛奶粥 + 水煮蛋                  │|
|  │ 热量: 320 kcal                       │|
|  │                                     [▶] │
|  └─────────────────────────────────────┘|
|                                           |
|  ┌─────────────────────────────────────┐|
|  │ 午餐                                │|
|  │ 香煎鸡胸肉 + 西兰花 + 杂粮饭         │|
|  │ 热量: 450 kcal  [专家推荐]         │|
|  │                                     [▶] │|
|  └─────────────────────────────────────┘|
|                                           |
|  ┌─────────────────────────────────────┐|
|  │ 晚餐                                │|
|  │ 蒸鳕鱼 + 蒜蓉青菜 + 紫薯            │|
|  │ 热量: 380 kcal                       │|
|  │                                     [▶] │|
|  └─────────────────────────────────────┘|
|                                           |
|  今日总热量: 1150 kcal                    |
|  建议范围: 1100-1300 kcal                 |
|                                           |
+-------------------------------------------+
|   首页        食谱库      我的            |
+-------------------------------------------+
```

### 设计要点
- 顶部导航：我的 / 今日推荐 / 食谱库
- 三餐卡片垂直排列，每餐显示：
  - 菜名（简短，如"燕麦牛奶粥 + 水煮蛋"）
  - 热量数值
  - 专家推荐标签（若有）
  - 右侧箭头，点击进入详情
- 底部显示今日总热量和建议范围
- 底部 Tab 导航

### 交互说明
- 点击某餐卡片 → 进入食谱详情页
- 卡片右滑或长按 → 弹出"换一个"选项
- 点击"食谱库" → 进入食谱库页面（V1.0功能，MVP可隐藏）

## 页面 4: 食谱详情页

### 布局

```
+-------------------------------------------+
|  < 返回       午餐详情        [ 收藏 ❤ ]   |
+-------------------------------------------+
|                                           |
|   香煎鸡胸肉 + 西兰花 + 杂粮饭            |
|   [专家推荐]                               |
|   预估热量: 450 kcal                      |
|                                           |
|  ┌─────────────────────────────────────┐|
|  │ 食材清单                            │|
|  │ • 鸡胸肉 150g                       │|
|  │ • 西兰花 200g                       │|
|  │ • 杂粮饭 150g                       │|
|  │ • 橄榄油 10ml                       │|
|  │ • 生抽、盐 适量                     │|
|  └─────────────────────────────────────┘|
|                                           |
|  ┌─────────────────────────────────────┐|
|  │ 营养成分                            │|
|  │ 热量: 450 kcal                      │|
|  │ 蛋白质: 35g                         │|
|  │ 碳水: 45g                           │|
|  │ 脂肪: 12g                           │|
|  └─────────────────────────────────────┘|
|                                           |
|  ┌─────────────────────────────────────┐|
|  │ 制作步骤                            │|
|  │ 1. 鸡胸肉切块，用生抽、盐腌制15分钟 │|
|  │ 2. 西兰花切小朵，焯水备用           │|
|  │ 3. 热锅下橄榄油，煎鸡胸肉至金黄    │|
|  │ 4. 加入西兰花翻炒均匀，出锅         │|
|  │ 5. 搭配杂粮饭即可                   │|
|  └─────────────────────────────────────┘|
|                                           |
|        [ 换一个 ]  [ 我要做这道菜 ]      |
|                                           |
|                                           |
|   参考文献：                              |
|   • 中国居民膳食指南（2022）              |
|   • 糖尿病膳食指南（2021）                |
|                                           |
+-------------------------------------------+
```

### 设计要点
- 顶部返回按钮 + 标题 + 收藏按钮
- 菜名 + 专家推荐标签 + 热量
- 食材清单（清晰列出）
- 营养成分表格（4项核心数据）
- 制作步骤（编号，简洁易懂）
- 底部两个按钮："换一个"、"我要做这道菜"
- 底部显示参考文献来源

### 交互说明
- 点击"换一个" → AI重新生成该餐次的其他食谱
- 点击"我要做这道菜" → 标记为今日已吃（V1.0功能，MVP可隐藏或仅视觉反馈）
- 点击"收藏" → 加入个人收藏（V1.0功能）
- 长按某个食材 → 弹出"不包含X，换一个"

## 底部导航栏设计

### MVP版本（3个Tab）

```
+---------+---------+---------+
|  我的  | 今日推荐| 食谱库  |
+---------+---------+---------+
```

### 设计说明
- **我的**：用户画像、收藏、设置
- **今日推荐**：首页，显示AI生成的今日三餐
- **食谱库**：浏览所有食谱（V1.0功能，MVP可隐藏该Tab或显示为灰度）

### 选中态样式
- 选中Tab：文字主色，图标实心
- 未选中Tab：文字灰色，图标空心

## 可访问性说明

1. **字体大小**：正文不小于14px，方便老年人阅读
2. **按钮点击区域**：不小于44x44px，符合小程序设计规范
3. **颜色对比度**：文字与背景对比度符合WCAG AA标准
4. **触控反馈**：所有可点击元素有按下状态
