# AI智能人群食谱小程序

## 项目概述

针对孕期、慢性病、健身、失眠等特定人群，提供个性化、科学的饮食建议和食谱推荐的微信小程序。

## 目标用户

| 用户画像 | 核心诉求 |
|---------|---------|
| 孕期女性 | 避免禁忌食物，补充叶酸、铁、钙，控制体重增长 |
| 慢性病患者 | 控制糖分/盐分/蛋白质摄入，稳定病情指标 |
| 健身人群 | 精确计算卡路里和宏量营养素，支持训练目标 |
| 失眠人群 | 避免刺激性食物，摄入助眠成分 |

## 核心价值

- **个性化**：根据用户画像AI生成专属食谱
- **科学性**：营养师审核内容，标注参考文献
- **轻量化**：用完即走，降低使用门槛
