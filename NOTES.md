# Project Notes — 小程序工厂

## [2026-03-15] PRD迭代核心坑位

1. **不要绑定具体项目**：生产线是通用机制，不因AI食谱被影响
2. **不要预估时间**：用户明确说"你的预估都不准"，里程碑只写内容和验收标准
3. **规范在M0中发现，不是预设的**：设计token、组件库等在第一个小程序中沉淀，不要提前臆想细节
4. **三带并行模型**：选题带（不限量）、开发带（1-2并发）、运营带（最多5）同时跑
5. **工具链分工**：Claude Code出文档，Codex做代码实现

## [2026-03-15] AI生成页面质量问题

用户反馈最大痛点：AI理解需求后开发出来的页面极丑/排版错乱/导航bug。
解决方案：设计系统（设计token + 组件库 + 布局模板 + 导航标准），在M0中建立。
